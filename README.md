## Как запустить проект?

### Способ 1 (Через Python)
1. Скачать [python](https://www.python.org/) 3.14.X и выше.
2. Скачать зависимости (pip install -r requirements.txt)
3. Создать .env, и вместо BOT_TOKEN засунуть ваш токен.
4. Запустить файл 

```bash
python main.py # Или python3 main.py
```
